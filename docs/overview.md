# Study Group App - 勉強会管理アプリ

## 概要

週一のエンジニア勉強会を支援するアプリケーション。  
メンバーの興味リスト管理、学習スケジュール管理、学習成果の共有を行う。

## 現在のメンバー

- しらおかさん
- ノリさん

## 機能（段階的に実装）

### Phase 1（MVP）
- **興味リスト管理**: メンバーごとの興味あるテーマの表示・追加・削除
- **学習予定管理**: 次回以降の勉強会テーマと日程の管理

### Phase 2（将来）
- **学習成果の共有**: 各回で学んだことのテキスト記録・閲覧

## 技術方針

- **外部ライブラリ不使用**: GHC 標準ライブラリ（base）のみで実装する（学習目的）
- **データ永続化**: JSON ファイルに保存（RDB は使わない）
- **コンテナ**: Docker（デプロイ・開発環境の統一）
- **ホスティング**: Google Cloud（要調査、コスト重視）

## アーキテクチャ

```
┌───────────┐     HTTP/JSON     ┌──────────────┐     ┌───────────┐
│  CLI (別)  │ ◄──────────────► │  Haskell API  │ ──► │ data.json │
└───────────┘                   └──────────────┘     └───────────┘
```

JSON パース・HTTP サーバー・ルーティングを全て自前で実装することで、
Haskell の基礎力（パーサー、IO、型設計）を実践的に鍛える。
